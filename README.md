# LeaderboardBot

Open-source Discord + OCR bot that tracks team wins/losses.
## Setup
1. Clone the repo
2. Install dependencies
```bash
npm install
```
3. Create a `config.json` file in the root directory
```json
{
    "token": "your token",
    "channel": "channel id",
    "team": "team name"
}
```
4. Run the bot
```bash
npm start
```
